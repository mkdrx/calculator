## Based on WebDevSimplified

- Clone the project

```bash
  git clone https://github.com/mkdrx/calculator.git
```

- Go to the project directory

```bash
  cd calculator
```

- Install dependencies

```bash
  npm install
```

- Run dev

```bash
  npm start
```
